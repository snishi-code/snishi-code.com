# snishi-code.com

[snishi-code.com](https://snishi-code.com) のトップ（入口）サイトです。
医療・個人向けに制作した、インストール不要で使えるウェブアプリ（PWA）を公開しています。

## 設計の考え方

すべてのアプリに共通する原則です。

- **データは端末内のみ** — ユーザーが入力したデータは利用者の端末内だけに保存し、
  外部サーバーへ送信しません（解析・トラッキングも入れていません）。
- **オフライン動作** — ネット接続がなくても全機能が使えるよう設計しています。
- **個人情報を持たない設計** — 医療アプリでも、氏名・ID・生年月日などの個人を
  特定できる情報を入力しない運用を前提にしています。

## このリポジトリの構成

入口サイトとアプリ本体は、別々のリポジトリ・別々のサブドメイン（独立した Origin）に
分かれています。

| サイト | 内容 | リポジトリ |
|---|---|---|
| [snishi-code.com](https://snishi-code.com) | 入口（このリポジトリ） | `snishi-code.com` |
| [medical.snishi-code.com](https://medical.snishi-code.com) | 医療向けアプリ | `snishi-code-medical` |
| [personal.snishi-code.com](https://personal.snishi-code.com) | 個人向けアプリ | `snishi-code-personal` |

このリポジトリ自体は静的な入口ページ（`index.html` / `shared.css` /
`site-links.js`）のみで、アプリのコードは含みません。
