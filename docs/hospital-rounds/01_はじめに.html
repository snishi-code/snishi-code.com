回診アプリ — はじめに

このアプリについて

回診時の患者情報（SOAP形式）をスマートフォンやタブレットで手軽に記録・管理するためのアプリです。

モバイル電子カルテのような高度なものではなく、名簿やメモ帳を置き換えるアプリです QRコードなどを利用し、純粋な事務作業を削減できるようにしています ブラウザで実際に触ってみて気に入ったら使用を検討してみてください

  • インターネット接続不要（完全オフライン動作）
  • 外部へデータを送信する機能は実装していません。ソースコードは GitHub で公開しています

QRコードリーダーを使った転記： スマートフォンのデフォルトカメラアプリでQRを読み取るとテキストとしてコピーできます。また、PCに接続したQRコードリーダー（バーコードリーダー）があれば、電子カルテの転記したい欄にカーソルを置いた状態で読み取ることで、キーボード入力と同じ動作でそのまま貼り付けられます。リーダーの接続可否は施設のシステム担当者にご確認ください。

ご利用にあたっての注意

  • 本アプリは個人が開発したものです。利用は自己責任でお願いします
  • 動作の正確性・安全性について、開発者は一切の責任を負いません
  • 医療機関での導入にあたっては、施設のセキュリティ担当者・情報管理部門の承認を得た上でご使用ください

PWAとは？

このアプリは PWA（Progressive Web App） という形式で作られています。

アプリストアからインストールする必要はなく、ブラウザで開いたページをそのままホーム画面に追加するだけで、普通のアプリのように使えます。Wi-Fi不要でオフラインでも動作します。

データの保存について

入力内容はリアルタイムでブラウザ内に自動保存されます。

ただし、これはブラウザ内部の保存であり、以下の場合にデータが失われます：

  • ブラウザの履歴・キャッシュを削除した
  • 端末を初期化した
  • 別のブラウザで開いた（ブラウザをまたいでデータは共有されません）

Wordで作業中にこまめに上書き保存するのと同じ感覚で、上矢印アイコン（保存）からJSONファイルとして定期的に保存しておくことを推奨します。
