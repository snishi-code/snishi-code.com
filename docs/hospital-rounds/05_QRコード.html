QRコード

概要

このアプリはインターネットを使わず、QRコードを介して2台の端末間でデータをやり取りできます。

主な用途： - 回診後のSOAPを別端末（ナースステーションのPCなど）に転送する - 主治医と担当看護師で情報を共有する

個別患者のQR（SOAP内容）

送信側

  1. 患者の詳細入力画面でステータスを「緑（回診済）」にする
  2. QRコードが自動表示されます
  3. データ量が多い場合は複数枚に分割されます（「前」「次」で切り替え）

受信側・電子カルテへの転記

QRコードを読み取るとテキストが得られます。以下の形式で出力されます。

（メモ内容）
――
(S)
患者の訴え内容
――
(O)
SpO2：98％  血圧：120/80mmHg  脈拍：72回  体温：36.5℃
肺音：明らかなラ音なし
腸音：正常
――
(A)
著変なし
――
(P)
現行加療継続

このテキストを電子カルテに転記して利用できます。

QRコードリーダーを使った転記： スマートフォンのデフォルトカメラアプリでQRを読み取るとテキストとしてコピーできます。また、PCに接続したQRコードリーダー（バーコードリーダー）があれば、電子カルテの転記したい欄にカーソルを置いた状態で読み取ることで、キーボード入力と同じ動作でそのまま貼り付けられます。リーダーの接続可否は施設のシステム担当者にご確認ください。

共有一覧のQR（複数患者まとめて）

送信側

  1. 各患者の詳細入力画面で「共有」欄に引き継ぎ内容を入力する
  2. 「共有」画面を開く
  3. 「QR表示」ボタンをタップ
  4. 複数QRに分割されている場合は「前」「次」で順番に表示

受信側

  1. スマートフォンのデフォルトカメラアプリでQRコードを読み取り、テキストをコピーする
  2. 「共有」画面を開く
  3. 「受信メモ」欄にテキストを貼り付けて内容を確認する

ヒント: 複数QRの場合は必ず順番（1/3 → 2/3 → 3/3）に読み取ってください。
