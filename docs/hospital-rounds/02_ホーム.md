# ホーム

## 概要

![[02_ホーム-1779021052489.webp|300x238]]

起動すると最初に表示される画面です。登録された患者がマス目（グリッド）状に並び、それぞれの色でステータスを一目で確認できます。

---

## アプリタイトル

画面上部のタイトル（初期値：「回診」）は直接タップして編集できます。担当者名や病棟名を入れて使い分けることができます。同じ端末を複数人で共有する運用も可能です。

患者数が多い場合は「201 テスト」のように患者名に部屋番号を併記する使い方が便利です。[部屋番号機能](08_部屋番号機能（オプション）.md)を使うことも可能です。

---

## ステータス

白・青以外は自由に使い方を工夫できます。以下は開発者が想定している一例です。

| 色   | 使用例 | 変更方法                                         |
| --- | --- | -------------------------------------------- |
| 白   | 未回診 | デフォルト                                        |
| 黄   | 保留  | 患者画面から                                       |
| 緑   | 回診済 | 患者画面から                                       |
| 灰   | 完了  | 患者画面から                                       |
| 青   | 追記  | データ取込時に自動（→ [05_データの取込・保存](05_データの取込・保存.md)） |

---

## ボタン説明

![[01_はじめに-1779020389587.webp|300x44]]

左から以下のボタンが並んでいます。

| ボタン       | 機能                                                                 |
| --------- | ------------------------------------------------------------------ |
| 赤バツ印（全消去） | 設定などを含む全情報を削除。先にデータを保存しておくことを推奨します                                 |
| バツ印（クリア）  | 患者の一部データを一括で削除。どのデータを削除するかは設定から指定できます。各患者画面からその患者のデータだけを削除することも可能。 |
| 下矢印（取込）   | 端末からデータを読み込む。                                                      |
| 上矢印（保存）   | 端末にデータを保存。                                                         |

---

## 患者ボタンをタップ

![[02_ホーム-1779021378251.webp|300x238]]

タップすると該当患者の詳細入力画面に移動します。

---

## 長押し（患者の追加・削除）

![[02_ホーム-1779021411548.webp|300x238]]

患者ボタンを長押しして放すと操作メニューが表示されます。

| 操作       | 内容                     |
| -------- | ---------------------- |
| 1人追加（下へ） | 選択した患者の下に新しい患者ボタンを1つ追加 |
| 5人追加（下へ） | 選択した患者の下に新しい患者ボタンを5つ追加 |
| 削除       | 選択した患者を削除（確認あり）        |

---

![[02_ホーム-1779021477607.webp|300x238]]
