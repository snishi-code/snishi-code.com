# 設定

![[06_設定-1779022896252.webp|300x238]]

ホーム画面のギアアイコン（設定）から開きます。

---

## 一覧

![[06_設定-1779023407112.webp|300x238]]

右上の表ボタンをタップすると一覧画面に移動します。紙カルテ使用施設などは印刷したものをハサミでカットしてカルテに貼り付けることができます。

本機能は今後改善予定です。

---

## デフォルト文言

S・A・P の各欄が空欄のときに出力される文言を設定します。

| 項目  | 初期値    |
| --- | ------ |
| S   | （空欄）   |
| A   | 著変なし   |
| P   | 現行加療継続 |

---

## O所見項目のカスタマイズ

身体所見（O）のチェックボタン項目を追加・編集・削除できます。

### 初期設定の項目

| 項目名     | 「正常」ボタン押下時の出力 |
| ------- | ------------- |
| General | 良好            |
| 肺音      | 明らかなラ音なし      |
| 腸音      | 正常            |
| 腹部      | 平坦軟、圧痛なし      |
| 食事      | 摂取良好          |
| 排泄      | 尿・便ともに特記なし    |

![[06_設定-1779023005778.webp|300x238]]

- ＋アイコンをタップすると新しい項目を追加できます
- 各項目のゴミ箱アイコンをタップすると該当項目を削除できます（確認あり）
- 回転矢印アイコン（初期化）をタップすると初期設定の6項目に戻ります（確認あり）

---

## ✕印（クリア対象のカスタマイズ）

![[06_設定-1779023271623.webp|300x238]]

詳細入力画面の✕丸アイコン（クリア）で消去する項目を選択します。赤く表示されている項目がクリアされます。タップすることでオン／オフを切り替えられます。

| 項目        | デフォルト |
| --------- | ----- |
| メモ        | OFF   |
| S         | ON    |
| O（バイタル含む） | ON    |
| A         | OFF   |
| P         | OFF   |
| 共有        | OFF   |
| 黄         | ON    |
| 緑         | ON    |
| 灰         | ON    |
| 青         | OFF   |
